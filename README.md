# Java Collections Framework

This repository contains Java programs and examples based on the Java Collections Framework. The purpose of this repository is to understand how data can be stored, managed, and manipulated efficiently using built-in collection classes.

## Topics Covered

### 1. ArrayList

* Creating an ArrayList
* Adding Elements
* Accessing Elements
* Updating Elements
* Removing Elements
* Iterating Through ArrayList

### 2. LinkedList

* Creating a LinkedList
* Adding and Removing Elements
* Traversing a LinkedList

### 3. Stack

* Push Operation
* Pop Operation
* Peek Operation
* Checking if Stack is Empty

### 4. Queue

* Adding Elements
* Removing Elements
* Peek Operation
* Queue Traversal

### 5. HashSet

* Creating a HashSet
* Adding Elements
* Removing Elements
* Avoiding Duplicate Values

### 6. LinkedHashSet

* Maintaining Insertion Order
* Set Operations

### 7. TreeSet

* Sorting Elements Automatically
* Set Operations

### 8. HashMap

* Storing Key-Value Pairs
* Adding Entries
* Accessing Values
* Updating Entries
* Removing Entries
* Iterating Through HashMap

### 9. LinkedHashMap

* Maintaining Insertion Order
* Key-Value Operations

### 10. TreeMap

* Sorted Key-Value Storage
* Map Operations

### 11. Iterator

* Traversing Collections
* Using Iterator Methods

### 12. Comparable

* Natural Sorting of Objects

### 13. Comparator

* Custom Sorting of Objects
